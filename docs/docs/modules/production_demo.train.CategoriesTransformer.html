production_demo.train.CategoriesTransformer¶

class production_demo.train.CategoriesTransformer(category_cols: list, n_buckets: int = 100000)[source]¶

Custom transformer for categories

Use this to create hashed numeric values for a list of categorical columns for any pandas dataframe. This can be used in place of One Hot Encoding of categorical values, and does not require storing a lookup of possible categories for each column.

The LightGBM package performs best without transforming into a high dimensional (i.e. one hot encoded) feature space; instead, we will directly train using hashed numeric buckets.
